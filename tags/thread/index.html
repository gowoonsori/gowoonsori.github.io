    멀티 스레딩

    백기선님의 유튜브로 진행하시는 스터디를 진행하며 올리는 정리 블로그입니다. 📌 Thread란? 쓰레드라는 개념을 이해하기 위해서는 프로세스와 병행성(Concurrency)의 개념을 알면 더 좋을 것같다. ◾ 프로세스란 프로세스의 사전적 의미는 컴퓨터에서 실행되고 있는 컴퓨터 프로그램을 말한다. 여러 개의 프로세서를 사용하는 것을 멀티 프로세싱이라고 하며, 같은 시간에 여러 개의 프로그램을 띄우는 시분할 방식을 멀티태스킹 이라고 한다. 특징 프로그램이 실행되는 단위 모든 프로그램은 실행 될 때 하나 이상의 프로세스를 갖는다. 각 프로세스는 다른 프로세스의 변수나 자료 구조에 접근할 수 없다. »
