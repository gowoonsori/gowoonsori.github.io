    ⚡ 문제링크 : 프로그래머스



    ✔ 문제

    효진이는 멀리 뛰기를 연습하고 있습니다. 효진이는 한번에 1칸, 또는 2칸을 뛸 수 있습니다. 칸이 총 4개 있을 때, 효진이는 (1칸, 1칸, 1칸, 1칸) (1칸, 2칸, 1칸) (1칸, 1칸, 2칸) (2칸, 1칸, 1칸) (2칸, 2칸) 의 5가지 방법으로 맨 끝 칸에 도달할 수 있습니다. 멀리뛰기에 사용될 칸의 수 n이 주어질 때, 효진이가 끝에 도달하는 방법이 몇 가지인지 알아내, 여기에 1234567를 나눈 나머지를 리턴하는 함수, solution을 완성하세요. 예를 들어 4가 입력된다면, 5를 return하면 됩니다.


    ❗ 제한사항

    n은 1 이상, 2000 이하인 정수입니다.

    ❗ 입출력 예

    n	result
4	5
3	3

    ❗ 입출력 예 설명

    입출력 예 #1
위에서 설명한 내용과 같습니다.

입출력 예 #2
(2칸, 1칸)
(1칸, 2칸)
(1칸, 1칸, 1칸)
총 3가지 방법으로 멀리 뛸 수 있습니다.



    ✔ 문제 풀이

    문제만 보면 재귀함수나 반복문을 이용하여 쉽게 풀 수 있을 것 같아 진행해보았습니다. 하면서도 시간오버 날 것 같은데.. 싶었는데 역시나 9번 이후로는 시간 초과가 발생하였습니다.

    재귀로 작성하면서도 DP를 이용하면 되겠다 싶어, 시간 오버 뜨자마자 DP로 바꾸어 풀어 주었습니다.


    📚 알고리즘

    • 칸이 1개일 때는 뛸 수 있는 방법은 1가지
    • 칸이 2개일 때 뛸 수 있는 방법은 2가지 (1,1) ,(2)
    • 칸이 3개일 때 뛸 수 있는 방법은
      • 처음에 1칸 뛰었을 때 = 칸이 2칸일 때의 가짓 수
      • 처음에 2칸 뛰었을 때 = 칸이 1칸일 때의 가짓 수
    • 따라서 칸이 n개 일때 뛸 수 있는 방법은 [n-1]개의 가짓 수 + [n-2]개의 가짓 수

    📃 코드 ( 주석 없는 코드 )

    #include <string>
#include <vector>

using namespace std;

long long solution(int n) {
    vector<long long> v;
    
    for(int i=0; i < n ; i++){
        if( v.size() < 3 ) v.push_back(i+1);       
        else v.push_back( (v[i-1] % 1234567) + (v[i-2] % 1234567) );    
    return (v.back()) % 1234567;
}



    📃 코드 ( 주석 있는 코드 )

    ( 접기 / 펼치기 ) 
    #include <string>
#include <vector>

using namespace std;

long long solution(int n) {
    vector<long long> v;
    
    for(int i=0; i < n ; i++){
        if( v.size() < 3 ) v.push_back(i+1);        //칸이 1 or 2칸 일때는 경우의 수 인 1, 2를 삽입 해준다
        else v.push_back( (v[i-1] % 1234567) + (v[i-2] % 1234567) );    //3칸 이상부터는 n-1칸 + n-2칸의 경우의 수를 더해준다.    
    }    
    return (v.back()) % 1234567;
}
