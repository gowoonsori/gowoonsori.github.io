    ✔ 문제


    ✔ 문제 풀이(틀린 풀이)

    Lv2 문제임에 불구하고 매우 쉽다고 생각을 하면서, 쉽게 코드를 작성 해나갔다.

    첫 행부터 각 행의 최댓 값을 구하고 최댓 값의 땅을 밟으며, 행을 중복해서 밟을 수 없으므로 다음 행을 0으로 바꿔주는 식으로 문제 풀이를 했다.


    📃 틀린 코드

    #include <iostream>
#include <vector>

using namespace std;

int solution(vector<vector<int> > land)
{
    int answer = 0;
    for(int i = 0; i < land.size() ; i++){
        int index , max = 0;
        for(int j =0; j < 4; j++){
            if(land[i][j] >= max){
                index = j;
                max = land[i][j];
            }
        }
        if(i != land.size()-1) land[i+1][index] = 0;
        answer += land[i][index];
    }
    return answer;
}

    결과:

    결과는 문제 예시의 테스트 케이스는 통과를 하였으나 그 외 모든 케이스는 실패지옥을 보여주었다. 왜 모두 실패를 했는지 곰곰히 생각을 해보니 너무 쉽게 생각하고 풀이한게 잘못이었다.

    예를 들어

    1 | 2 | 3 | 5
5 | 6 | 7 | 100
4 | 3 | 2 | 1

    의 경우에는 3 -> 100 -> 4 를 밟아 107 이 최대값인데 내 풀이대로라면, 5 -> 7 -> 4 를 밟으며 16을 도출해내는 것이다.

    따라서, 해당 문제는 모든 경우의 수에서의 최대값을 구하면 된다.



    ✔ 문제 풀이 (정답)

    DP (Dynamic Programming)(동적 계획법)을 이용하여 풀이를 해주면 된다.


    DP 란?

    문제를 여러 개의 문제로 나누어 푸는 방법이다. 분할 정복과 비슷하며, 분할 정복과 다른 점은 작은 부분의 결과 값을 메모(저장)해놓고 그 보다 큰 부분을 풀어나갈때 앞서 메모해놓은 결과값을 이용하여 풀이하는 점이 다릅니다.

    [DP 참고 자료] (https://galid1.tistory.com/507)


    📚 알고리즘

    각 행의 땅마다 갈 수 있는 최대값을 구하여, 마지막 행에서 최댓 값을 갖는 땅이 정답이 된다.

    예를 들어, 아래와 같은 케이스라면

    1 | 2 | 3 | 5
5 | 6 | 7 | 100
4 | 3 | 2 | 1

    1 | 2 | 3 | 5
10| 11| 12| 103      
107|106|105| 13

    이 되어 최대 값은 107이 됩니다.


    📃 코드 ( 주석 없는 코드 )

    #include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int solution(vector<vector<int> > land)
{
    int answer = 0;
    for(int i = 0; i < land.size()-1 ; i++){
        land[i+1][0] += max(land[i][1],max(land[i][2],land[i][3]));  
        land[i+1][1] += max(land[i][0],max(land[i][2],land[i][3]));
        land[i+1][2] += max(land[i][0],max(land[i][1],land[i][3]));
        land[i+1][3] += max(land[i][0],max(land[i][1],land[i][2]));
    }
    answer = max(land[land.size()-1][0], max(land[land.size()-1][1], max( land[land.size()-1][2], land[land.size()-1][3] )));       
    return answer;
}



    📃 코드 ( 주석 있는 코드 )

    ( 접기 / 펼치기 ) 
    #include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int solution(vector<vector<int> > land)
{
    int answer = 0;
    
    /*각 항의 바로 다음 아래로는 갈 수 없으니 같은 항을 제외한 값 중에서 가장 큰 값을 찾아 더하여 최댓값을 구한다*/
    for(int i = 0; i < land.size()-1 ; i++){
        land[i+1][0] += max(land[i][1],max(land[i][2],land[i][3]));
        land[i+1][1] += max(land[i][0],max(land[i][2],land[i][3]));
        land[i+1][2] += max(land[i][0],max(land[i][1],land[i][3]));
        land[i+1][3] += max(land[i][0],max(land[i][1],land[i][2]));
    }

    
    /*마지막 줄에 있는 값들이 최종 결과 값이기 때문에 마지막 줄에서 최대값을 구하면 그 값이 answer이 된다. */
    answer = max(land[land.size()-1][0], max(land[land.size()-1][1], max( land[land.size()-1][2], land[land.size()-1][3] )));
    return answer;
}
