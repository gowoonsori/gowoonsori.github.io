    String нғҖмһ…мқ„ м„ м–ён•ҳлҠ” л°©лІ•

    в—ҫ лҰ¬н„°лҹҙ л°©мӢқ

    String str = "Hello";
String str2 = "Hello";
System.out.println(str == str2);  //true

    нҺён•ҳкё° л•Ңл¬ём—җ л§Һмқҙ мӮ¬мҡ©н•ҳлҠ” л°©лІ•мңјлЎң нҒ°л”°мҳҙн‘ң(" ")лЎң л°”лЎң н• лӢ№н•ҳлҠ” л°©лІ•мқҙлӢӨ.

    мқҙ л°©лІ•мқҖ лӮҙл¶Җм ҒмңјлЎң JVMл©”лӘЁлҰ¬м—җ мһҲлҠ” мғҒмҲҳн’Җм—җ м ҖмһҘмқҙ лҗҳлҠ”лҚ° мқҙл•Ңл¬ём—җ к°ҷмқҖ л¬ёмһҗм—ҙмқ„ лӢӨлҘё ліҖмҲҳм—җ м„ м–ёмқ„ н•ҳкі  ==м—°мӮ°мқ„ мҲҳн–үн•ҳл©ҙ trueк°Җ лӮҳмҳӨлҠ” мқҙмң к°Җ лҗңлӢӨ. лӮҙл¶Җм ҒмңјлЎң мғҒмҲҳн’Җм—җ л“ұлЎқлҗң л¬ёмһҗм—ҙмқҳ referenceлҘј internмқ„ мқҙмҡ©н•ҙ нҳём¶ңн•ҙ н• лӢ№н•ҳкё° л•Ңл¬ём—җ к°ҷмқҖ л¬ёмһҗм—ҙмқ„ м°ёмЎ°н•ҳкі  мһҲкё° л•Ңл¬ёмқҙлӢӨ.

    в—ҫ мғқм„ұмһҗ л°©мӢқ

    StringлҸ„ classмқҙкё° л•Ңл¬ём—җ мғқм„ұмһҗлЎң м„ м–ёмқҙ к°ҖлҠҘн•ҳкі  мғқм„ұмһҗлЎң м„ м–ён•ҳкІҢ лҗҳл©ҙ м„ңлЎң лӢӨлҘё к°қмІҙк°Җ л°ҳнҷҳмқҙлҗҳкё° л•Ңл¬ём—җ к°ҷмқҖ л¬ёмһҗм—ҙмқҙлҚ”лқјлҸ„ ==м—°мӮ°мӢң falseк°Җ м¶ңл ҘлҗңлӢӨ.

    byte[] bytes = { 72, 101, 108, 108, 111 };
String s = new String(bytes);
System.out.println(s);  //Hello
String s1 = new String("Hello");
System.out.println(s == s1);  //false

    String ClassлҠ” лӮҙл¶Җм ҒмңјлЎң byte[] нҳ•нғңлЎң л¬ёмһҗм—ҙмқ„ м ҖмһҘн•ҳкі  кҙҖлҰ¬н•ҳкё° л•Ңл¬ём—җ мғқм„ұмһҗлҸ„ byte/char л°°м—ҙнҳ•нғңлЎң мЈјл©ҙ Stringмқҙ мғқм„ұмқҙ лҗҳкі  мғқм„ұмһҗ м•Ҳм—җ ""лЎң л¬ёмһҗм—ҙмқ„ мһ…л Ҙн•  мҲҳлҸ„ мһҲлӢӨ.

    мқҙл•Ң л¬ёмһҗм—ҙмқҖ мғҒмҲҳн’Җм—җм„ң н•ҙлӢ№ л¬ёмһҗм—ҙмқ„ ліөмӮ¬н•ҳм—¬ byte[]лЎң м ҖмһҘн•ҳкІҢ лҗҳлҠ”лҚ°, мқҙлҠ” лҰ¬н„°лҹҙ л°©мӢқкіј лҳ‘к°ҷмқҖ л°©мӢқмңјлЎң лҸҷмһ‘н•ҳкё° л•Ңл¬ём—җ көімқҙ л¬ёмһҗм—ҙмқ„ мқёмҲҳлЎңн•ҳлҠ” мғқм„ұмһҗ л°©мӢқмқҖ мҲҳн–үн•  н•„мҡ”к°Җ м—ҶлӢӨ.

    String s1 = new String("Hello").intern();
String s2 = new String("Hello");
System.out.println(s1 == s2.intern());  //true

    мғқм„ұмһҗ мғқм„ұ л°©мӢқлҸ„ intern()н•ЁмҲҳлҘј мқҙмҡ©н•ҳл©ҙ л¬ёмһҗм—ҙ л№„көҗк°Җ trueк°Җ лҗ  мҲҳ мһҲлӢӨ.

    лӮҙл¶Җм—җ м„ м–ёлҗҳм–ҙмһҲлҠ” byte[]к°Җ finalлЎң м„ м–ёмқҙ лҗҳм–ҙмһҲм–ҙ ліҖкІҪмқҙ л¶Ҳк°ҖлҠҘн•ҳлӢӨ.

    мҡ°лҰ¬к°Җ нқ”нһҲ м“°лҠ” +=л°©мӢқмқҳ л¬ёмһҗм—ҙмқ„ л¶ҷмқҙлҠ” кІғмқҖ лӮҙл¶Җм ҒмңјлЎңліҙл©ҙ мғҲлЎңмҡҙ к°қмІҙлҘј мғқм„ұн•ҙ л°ҳнҷҳн•ҳлҠ” л°©мӢқмқҙлӢӨ. к·ёлһҳм„ң л¬ёмһҗм—ҙмқ„ л¶ҷмқҙлҠ”лҚ° StirngBuilderлҘј м“°лҠ” кІғмқҙ мҶҚлҸ„к°Җ лҚ” л№ лҘё мқҙмң мқёлҚ° StringBuilderлҠ” лӮҙл¶Җм—җ кҙҖлҰ¬н•ҳлҠ” л¬ёмһҗм—ҙмқҙ finalмқҙ м•„лӢҲлқј лҸҷм ҒмңјлЎң мӮ¬мқҙмҰҲлҘј кҙҖлҰ¬н•ҳкё° л•Ңл¬ём—җ к·ёлҢҖлЎң л°°м—ҙм—җ мқҙм–ҙм“°кё° л•Ңл¬ёмқҙлӢӨ.

    String к°қмІҙмқҳ л¬ёмһҗм—ҙмқҖ л¶ҲліҖм„ұмқҙкё° л•Ңл¬ём—җ м•„лһҳм—җ кё°мҲ н•ҳлҠ” лҢҖл¶Җл¶„мқҳ л©”м„ңл“ңл“ӨмқҖ н•ҙлӢ№ л¬ёмһҗм—ҙмқ„ ліҖнҷҳн•ҳлҠ” кІғмқҙ м•„лӢҲлқј кё°мӨҖ л©”м„ңл“ңлҘј ліөмӮ¬н•ҳм—¬ ліҖнҷҳн•ҙм„ң мғҲлЎңмҡҙ л¬ёмһҗм—ҙмқ„ л°ҳнҷҳн•ҳлҠ” л°©мӢқмқҙкё° л•Ңл¬ём—җ лӘЁл‘җ Stringнҳ•мқ„ returnн•ҳлҠ” л°©мӢқмқёкІғмқ„ ліј мҲҳ мһҲлӢӨ.



    мһҗмЈј мӮ¬мҡ©н•ҳлҠ” л©”м„ңл“ң

    в—ҫ equals()

    String s1 = new String("Hello");
String s2 = new String("Hello");
System.out.println(s1 == s2);  //false
System.out.println(s1.equals(s2));  //false

    ==мҷҖ лӢӨлҘҙкІҢ Stringк°қмІҙ лӮҙл¶Җмқҳ л¬ёмһҗм—ҙ(byte[]) к°’мқ„ л№„көҗн•ҳлҠ” л©”м„ңл“ңмқҙлӢӨ.

    public static boolean equals(byte[] value, byte[] other) {
        if (value.length == other.length) {
            for (int i = 0; i < value.length; i++) {
                if (value[i] != other[i]) {
                    return false;
                }
            }
            return true;
        }
        return false;
    }

    лӮҙл¶Җн•ЁмҲҳлҘј ліҙл©ҙ Stringмқҙ byteл°°м—ҙмқё м җ мқ„ мқҙмҡ©н•ҙ н•ңкёҖмһҗм”© л№„көҗн•ҳкі  мһҲлҠ” кІғмқ„ ліј мҲҳ мһҲлӢӨ.

    в—ҫ compareTo(String anotherString)

System.out.println(s1.compareTo(s5));  //-1

    л‘җ л¬ёмһҗм—ҙмқ„ л№„көҗн•ҳм—¬ мӮ¬м „мҲңмңјлЎң кё°мӨҖ л¬ёмһҗм—ҙмқҙ лҚ”нҒ¬л©ҙ м–‘мҲҳ, м•„лӢҲл©ҙ мқҢмҲҳ, к°ҷмңјл©ҙ 0мқ„ л°ҳнҷҳн•ңлӢӨ. (мқҙл•Ң мӮ¬м „мҲңмқҖ кІ°көӯ asciiмҪ”л“ң к°’мқҙлӢӨ.)

    лӘЁл‘җ лҳ‘к°ҷлӢӨк°Җ s1,s5мҷҖ к°ҷмқҙ н•ңк°ңк°Җ л¬ёмһҗм—ҙмқҳ кёёмқҙк°Җ лҚ” кёёлӢӨл©ҙ л¬ёмһҗм—ҙмқҙ н•ңк°ңлқјлҸ„ лҚ”мһҲлҠ” s5к°Җ лҚ” нҒ¬лӢӨ.

    • м–‘мҲҳ : кё°мӨҖ л¬ёмһҗм—ҙмқҙ лҚ” нҒ¬лӢӨ.
    • мқҢмҲҳ : кё°мӨҖ л¬ёмһҗм—ҙмқҙ лҚ” мһ‘лӢӨ.
    • 0 : к°ҷлӢӨ.

    Comparable, ComparatorлҘј м •мқҳн•ҳлҠ”лҚ°лҸ„ л§Һмқҙ мӮ¬мҡ©лҗңлӢӨ.



    л¬ёмһҗм—ҙ м •л ¬

    в—ҫ мҳӨлҰ„м°ЁмҲң

    String s = "hello";

char[] array = s.toCharArray();
Arrays.sort(array);
s = new String(array);
System.out.println(s); //ehllo

    Arrays.sort()к°Җ кё°ліёмқҙ мҳӨлҰ„м°ЁмҲңм •л ¬мқёкұё мқҙмҡ©н•ҙм„ң toCharArray()лЎң char[]мқ„ л§Ңл“Өм–ҙ л¬ёмһҗм—ҙмқ„ м •л ¬н•  мҲҳ мһҲлӢӨ.


    в—ҫ лӮҙлҰјм°ЁмҲң

    String s = "hello";

String[] array = s.split("");
Arrays.sort(array, Collections.reverseOrder());
//Collections.reverse(Arrays.asList(array)); //мң„ л°©мӢқкіј к°ҷмқҖ кІ°кіјлҘј ліҙм—¬мӨҖлӢӨ.
s = String.join("",array);
System.out.println(s);  //ollhe

    л¬ёмһҗм—ҙмқ„ лӮҙлҰјм°ЁмҲңмңјлЎң м •л ¬н•ҳкі мһҗ мң„ л°©лІ•м—җм„ң Arrays.sort()м—җ ComparatorмҶҚм„ұмңјлЎң Collections.reverseOrder()лҘј мЈјкІҢ лҗҳл©ҙ arrayк°Җ Objectк°Җ м•„лӢҢ char[]нҳ•мқҙлқј м»ҙнҢҢмқј м—җлҹ¬к°Җ лӮңлӢӨ.

    Stringнҳ• л°°м—ҙмқ„ м„ м–ён•ҙ split("")мңјлЎң к°Ғ л¬ёмһҗл§ҲлӢӨ лӮҳлҲ„м–ҙ л°°м—ҙм—җ м ҖмһҘн•ҳкі  м •л ¬мқ„ мҲҳн–үн•ҳкі  join()мқ„ мқҙмҡ©н•ҙ л°°м—ҙмқ„ л¶ҷм—¬мЈјл©ҙ л¬ёмһҗм—ҙмқ„ лӮҙлҰјм°ЁмҲңмңјлЎң м •л ¬мқҙ к°ҖлҠҘн•ҳлӢӨ.
