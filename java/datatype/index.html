    л°ұкё°м„ лӢҳмқҳ мң нҠңлёҢлЎң м§„н–үн•ҳмӢңлҠ” мҠӨн„°л””лҘј м§„н–үн•ҳл©° мҳ¬лҰ¬лҠ” м •лҰ¬ лё”лЎңк·ёмһ…лӢҲлӢӨ.


    мһҗл°”м—җм„ң лҚ°мқҙн„° нғҖмһ…мқҖ нҒ¬кІҢ мӣҗмӢң(Primitive) нғҖмһ…кіј м°ёмЎ°(Reference) нғҖмһ…мқҙ мһҲлӢӨ.


    Primitive Type

    м •мҲҳ, мӢӨмҲҳ, л¬ёмһҗ, л…јлҰ¬ лҰ¬н„°лҹҙкіј к°ҷмқҖ мӢӨм ң лҚ°мқҙн„° к°’мқ„ м ҖмһҘн•ҳлҠ” нғҖмһ…

    мў…лҘҳ лҚ°мқҙн„° нғҖмһ… нҒ¬кё°(byte) кё°ліё к°’ лІ”мң„
    л…јлҰ¬нҳ• boolean 1byte false true, false
    л¬ёмһҗнҳ• char 2byte \u0000 0 ~ 65,535
    м •мҲҳнҳ• byte 1byte 0 -128~ 127
    м •мҲҳнҳ• short 2byte 0 -2^15 ~ 2^15 - 1
    м •мҲҳнҳ• int 4byte 0 -2^31 ~ 2^31 - 1
    м •мҲҳнҳ• long 8byte 0L -2^63 ~ 2^63 - 1
    мӢӨмҲҳнҳ• float 4byte 0.0F 1.4E-45 ~ 3.4028235E38
    мӢӨмҲҳнҳ• double 8byte 0.0 4.9E-324 ~ 1.7976931348623157E308

    в—ҫ boolean

    cмҷҖ к°ҷмқҙ trueмҷҖ falseлҘј н‘ңнҳ„н•ҳлҠ”лҚ° 1bitл§Ң мһҲмңјл©ҙ лҗ  кІғ к°ҷм§Җл§Ң JavaлҠ” лҚ°мқҙн„° мөңмҶҢлІ”мң„к°Җ 1Byteмқҙкё° л•Ңл¬ём—җ booleanлҸ„ 1byteмқҳ нҒ¬кё°лҘј к°–лҠ”лӢӨ.

    в—ҫ char

    cмқҳ кІҪмҡ° 1byteмқҳ нҒ¬кё°лҘј к°ҷм§Җл§Ң JavaлҠ” мң лӢҲмҪ”л“ңлҘј н‘ңнҳ„н•ҳкё° мң„н•ҙ кё°ліё 2byteлҘј к°–мңјл©°, Javaмқҳ мӣҗмӢң лҚ°мқҙн„°нғҖмһ…мӨ‘ мң мқјн•ҳкІҢ unsignedнҳ•нғңмқҙлӢӨ.

    в—ҫ м •мҲҳнҳ•

    JVMмқҳ мҠӨнғқмқҙ кё°ліём ҒмңјлЎң н”јм—°мӮ°мһҗлҘј 4byte лӢЁмң„лЎң м ҖмһҘн•ҳкё° л•Ңл¬ём—җ intліҙлӢӨ мһ‘мқҖ мһҗлЈҢнҳ•мқҳ к°’мқ„ кі„мӮ°н• л•Ң intнҳ•мңјлЎң нҳ•ліҖнҷҳлҗҳм–ҙм„ң м—°мӮ°мқҙ мҲҳн–үлҗҳкё° л•Ңл¬ём—җ көімқҙ byteмҷҖ shortнҳ•мқ„ мӮ¬мҡ©н•  н•„мҡ”лҠ” м—Ҷм–ҙ ліҙмқёлӢӨ.

    JVMмқҖ intнҳ•мқ„ м •мҲҳнҳ•мқҳ кё°ліё лҚ°мқҙн„° нғҖмһ…мңјлЎң мӮ¬мҡ©н•ҳкё° л•Ңл¬ём—җ longмқ„ мӮ¬мҡ©н•ҳкі мһҗ н•ңлӢӨл©ҙ л’Өм—җ lorLмқ„ л¶ҷм—¬м•ј н•ңлӢӨ.

    Java8 мқҙнӣ„л¶Җн„°лҠ” IntegerмҷҖ Longмқҳ WrapperнҒҙлһҳмҠӨм—җ unsigned кҙҖл Ё л©”мҶҢл“ңк°Җ м¶”к°Җлҗҳм–ҙ unsigned нҳ•нғңлҸ„ мқҙмҡ©мқҙ к°ҖлҠҘн•ҳлӢӨ.


    в—ҫ мӢӨмҲҳнҳ•

    doubleнҳ•мқҙ кё°ліё лҚ°мқҙн„° нғҖмһ…мңјлЎң floatнҳ•мқ„ мӮ¬мҡ©н•ҳкі мһҗ н•ңлӢӨл©ҙ л’Өм—җ f or FлҘј л¶ҷм—¬ мЈјм–ҙм•ј н•ңлӢӨ.

    л©”лӘЁлҰ¬ нҒ¬кё°лҠ” int,longкіј к°ҷм§Җл§Ң мӢӨмҲҳнҳ•мқҖ л¶ҖлҸҷмҶҢмҲҳм җ л°©мӢқмңјлЎң м ҖмһҘн•ҳм—¬ лҚ” л§ҺмқҖ лІ”мң„лҘј м ҖмһҘ н•  мҲҳ мһҲлӢӨ.

    floatмқҳ кІҪмҡ° л¶Җнҳё(1bit)+м§ҖмҲҳ(8bit)+к°ҖмҲҳ(23bit), doubleмқҳ кІҪмҡ° л¶Җнҳё(1bit)+м§ҖмҲҳ(11bit)+к°ҖмҲҳ(52bit)лЎң лӮҳлҲ„м–ҙ м ҖмһҘн•ҳкІҢ лҗҳм–ҙ н‘ңнҳ„лІ”мң„к°Җ кі м • мҶҢмҲҳм җл°©мӢқліҙлӢӨ л„“м§Җл§Ң мӢӨмҲҳ н‘ңнҳ„м—җ мһҲм–ҙ к·јмӮ¬м№ҳлҘј н‘ңнҳ„н•ҳкё° л•Ңл¬ём—җ мҳӨм°Ёк°Җ мЎҙмһ¬н•ҳлҠ” лӢЁм җмқ„ к°–кі мһҲлӢӨ.

    (0.1мқ„ 1000лІҲ лҚ”н•ҳкІҢ лҗҳл©ҙ 100мқҙ лҗҳм–ҙм•ј н•ҳм§Җл§Ң 100.09999… мҷҖ к°ҷмқҖ к·јмӮ¬к°’мқҙ м¶ңл ҘлҗңлӢӨ.)

    floatмқҳ кІҪмҡ° мҶҢмҲҳм җ м•„лһҳ 9мһҗлҰ¬, doubleмқҳ кІҪмҡ° 18мһҗлҰ¬к№Ңм§Җ н‘ңнҳ„мқҙ к°ҖлҠҘн•ҳлӢӨ.

    к°ҷмқҖ лҚ°мқҙн„° нғҖмһ…м—җм„ңлҸ„ int,long / float,doubleнҳ•мңјлЎң лӮҳлҲҲ мқҙмң лҠ” л©”лӘЁлҰ¬лҘј нҡЁмңЁм ҒмңјлЎң мӮ¬мҡ©н•ҳкё° мң„н•ҙм„ң мқҙлӢӨ.

    лӢӯ мһЎлҠ”лҚ° мҶҢ мһЎлҠ” м№јмқ„ мӮ¬мҡ©н•ҳлҠ” кІғмқҖ лӮӯл№„мқҙкё° л•Ңл¬ёмқҙлӢӨ.



    Refrerence Type

    мҳҒм–ҙ мқҳлҜё к·ёлҢҖлЎң м°ёмЎ° нғҖмһ…мңјлЎң м ҖмһҘлҗҳлҠ” к°’мқҙ мӣҗмӢңлҚ°мқҙн„°мҷҖ лӢӨлҘҙкІҢ мӢӨм ң к°’мқҙ м•„лӢҢ л©”лӘЁлҰ¬ мЈјмҶҢк°’мқҙ м ҖмһҘлҗҳлҠ” нғҖмһ…мңјлЎң л¬ёмһҗм—ҙ,л°°м—ҙ,enum,class,interfaceл“ұмқҙ мһҲлӢӨ.

    JVMмқҳ Runtime Data AreaмӨ‘ м°ёмЎ° нғҖмһ… ліҖмҲҳлҠ” лҹ°нғҖмһ„ мҠӨнғқ мҳҒм—ӯм—җ мӢӨм ң к°қмІҙлҠ” нһҷ мҳҒм—ӯм—җ м ҖмһҘлҗҳм–ҙ к°қмІҙлҘј мӮ¬мҡ©н• л•Ң л§ҲлӢӨ м°ёмЎ° ліҖмҲҳм—җ м ҖмһҘлҗң к°қмІҙмқҳ мЈјмҶҢлҘј л¶Ҳлҹ¬мҷҖ мӮ¬мҡ©н•ңлӢӨ.

    мң„м—җм„ң л§җн•ңкІғкіј к°ҷмқҙ мӢӨм ң лҚ°мқҙн„°лҠ” нһҷ мҳҒм—ӯм—җ м ҖмһҘлҗҳл©° лҚ°мқҙн„° нҒ¬кё°лҠ” м •н•ҙм ёмһҲм§Җ м•Ҡкі  лҸҷм ҒмңјлЎң н• лӢ№лҗҳл©° м°ёмЎ°н•ҳлҠ” ліҖмҲҳк°Җ м—ҶлӢӨл©ҙ Garbage Collectorк°Җ м ңкұ°н•ҳм—¬ л©”лӘЁлҰ¬лҘј кҙҖлҰ¬н•ңлӢӨ.

    в—ҫ Boxing : мӣҗмӢң нғҖмһ…мқ„ м°ёмЎ° нғҖмһ…мңјлЎң ліҖнҷҳмӢңнӮӨлҠ” кІғ

    в—ҫ Unboxing : м°ёмЎ° нғҖмһ…мқ„ мӣҗмӢң нғҖмһ…мңјлЎң ліҖнҷҳ мӢңнӮӨлҠ” кІғ

    Auto Boxingкё°лҠҘмңјлЎң лӘ…мӢңм ҒмңјлЎң мһ‘м„ұн•ҳм§Җ м•Ҡм•„лҸ„ мһҗлҸҷмңјлЎң Boxing мқ„ н•ҙмӨҖлӢӨ.

    int i= 1;
Integer integer = i; //Auto Boxing

    в—ҫ нҠ№м§•

    • nullмқ„ нҸ¬н•Ён•  мҲҳ мһҲлӢӨ.

      int example = null; //error
Integer integer = null;

    • м ңл„ӨлҰӯ нғҖмһ…м—җм„ң мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ

      List<int> list; //error
List<Integer> list;

    • лҚ°мқҙн„° м ‘к·ј мҶҚлҸ„к°Җ мӣҗмӢңнғҖмһ…м—җ л№„н•ҙ лҠҗлҰ¬лӢӨ

      мӣҗмӢңнғҖмһ…мқҖ мҠӨнғқм—җм„ң л°”лЎң мӮ¬мҡ©мқҙк°ҖлҠҘн•ҳлӮҳ м°ёмЎ°нғҖмһ…мқҖ мҠӨнғқм—җм„ң л©”лӘЁлҰ¬мЈјмҶҢлҘј к°–кі  нһҷм—җ м ‘к·јн•ҳлҠ” л°©лІ•мңјлЎң к°’мқ„ н•„мҡ”н• л•Ңл§ҲлӢӨ Unboxingмқ„ кұ°міҗм•јн•ҳкё° л•Ңл¬ём—җ м ‘к·јмҶҚлҸ„к°Җ лҠҗл Өм§ҖкІҢ лҗңлӢӨ.

      н•ҳм§Җл§Ң, нҒ° нҒ¬кё°мқҳ лҚ°мқҙн„° м ‘к·јмқҙ м•„лӢҢ м „лӢ¬,ліөмӮ¬мқҳ кІҪмҡ°м—җлҠ” мӣҗмӢңнғҖмһ…ліҙлӢӨлҠ” м°ёмЎ°нғҖмһ…мқҙ мўӢмқ„ мҲҳлҸ„ мһҲлӢӨ.



    лҰ¬н„°лҹҙ

    л©”лӘЁлҰ¬м—җ м ҖмһҘлҗҳм–ҙ ліҖн•ҳм§Җ м•ҠлҠ” к°’мқ„ лң»н•ҳл©° м»ҙнҢҢмқј нғҖмһ„м—җ м •мқҳлҗҳм–ҙ к·ёлҢҖлЎң мӮ¬мҡ©н•ҳлҠ” к°’

    лҢҖмһ… м—°мӮ°м—җм„ң лӘЁл“  мҡ°н•ӯмқҳ к°’л“Өмқ„ ліҙнҶө лҰ¬н„°лҹҙмқҙлқјкі  л¶ҖлҘёлӢӨ.

    boolean isTrue = true;
char c = 'C';
int i = 1001;
long il = 1001L;
float f = 1.1234F;
double d = 1.1234;
String str = "hello";

    мҷҖ к°ҷмқҙ мҡ°н•ӯм—җ мһҲлҠ” лӮҙк°Җ м •мқҳн•ң к°’л“ӨлЎң ліҖн•ҳм§Җ м•ҠлҠ” к°’л“Өмқ„ мқҳлҜён•ңлӢӨ.

    int i = 100_000_000;

    Java 8 мқҙнӣ„м—җмқҖ лҰ¬н„°лҹҙм—җ _лҘј кө¬л¶„мһҗлЎң мӮ¬мҡ©н•ҳм—¬ нҒ° мҲ«мһҗмӮ¬мқҙм—җ ,мқ„ м°Қм–ҙ ліҙкё°нҺён•ҳкІҢ н•ҳлҠ” кІғмІҳлҹј мқҙмҡ©мқ„ н•  мҲҳ мһҲлӢӨ.



    ліҖмҲҳ м„ м–ё л°Ҹ мҙҲкё°нҷ” л°©лІ•

    в—ҫ ліҖмҲҳ м„ м–ё

    (лҚ°мқҙн„° нғҖмһ…) (ліҖмҲҳлӘ…1),(ліҖмҲҳлӘ… 2)... ;

int i;
int j,k,l;

    мң„мҷҖ к°ҷмқҙ н•ҳлӮҳмқҳ ліҖмҲҳлҘј м„ м–ёмқҙ к°ҖлҠҘн•ҳкі  лҸҷмӢңм—җ м—¬лҹ¬к°ңлҸ„ к°ҖлҠҘн•ҳлӢӨ.


    в—ҫ ліҖмҲҳ мҙҲкё°нҷ”

    • м„ м–ё нӣ„ мҙҲкё°нҷ”

      int i;
i =10;

    • м„ м–ёкіј лҸҷмӢңм—җ мҙҲкё°нҷ”

      int i=10;



    ліҖмҲҳмқҳ мҠӨмҪ”н”„мҷҖ лқјмқҙн”„нғҖмһ„

    ліҖмҲҳмқҳ мҠӨмҪ”н”„лһҖ н•ҙлӢ№ ліҖмҲҳм—җ м ‘к·јн•  мҲҳ мһҲлҠ” лІ”мң„лҘј лӮҳнғҖлӮҙлҠ” кІғмңјлЎң мӨ‘кҙ„нҳё {}лҘј нҶөн•ҙ мҠӨмҪ”н”„лҘј лӮҳлҲ„лҠ” лё”лЎқ мҠӨмҪ”н”„мқҙкі  н•ҙлӢ№ ліҖмҲҳк°Җ м–ём ңк№Ңм§Җ мЎҙмһ¬н•ҳлҠ”м§Җк°Җ лқјмқҙн”„ нғҖмһ„мқҙлӢӨ.

    м§Җм—ӯліҖмҲҳмқҳ лқјмқҙн”„нғҖмһ„кіј мҠӨмҪ”н”„лҠ” мҪ”л“ң лё”лҹӯм—җ көӯн•ңлҗңлӢӨ.

    public class example {
    public static void main(String[] args){
        int i =1;
        if(true){
            int j=2;
            System.out.println(i);
        }
        System.out.println(j); //error
    }
}

    мң„мқҳ мҪ”л“ңм—җм„ң ifлӮҙм—җм„ң iлҘј м ‘к·јн• л•Ң ifмқҳ мҪ”л“ң лё”лҹӯм—җм„ң iлҘј м°ҫкі  м—Ҷмңјл©ҙ к·ёліҙлӢӨ мғҒмң„ лё”лҹӯмқё class лё”лҹӯм—җм„ң iлҘј м°ҫм•„ м ‘к·јмқҙ к°ҖлҠҘн•ҳм§Җл§Ң jлҠ” ifлӮҙм—җ м„ м–ёлҗҳм–ҙ ifмқҳ лё”лҹӯ({})м—җ көӯн•ңлҗҳкё° л•Ңл¬ём—җ ifл°–м—җ лӮҳмҷҖм„ңлҠ” м ‘к·јмқҙ л¶Ҳк°ҖлҠҘн•ҳкі  ifмқҳ лё”лҹӯмқҙ лҒқлӮҳл©ҙм„ң ifм—җ көӯн•ңлҗҳм–ҙмһҲлҚҳ м§Җм—ӯ ліҖмҲҳл“ӨмқҖ мҠӨнғқмҳҒм—ӯм—җм„ң мҶҢл©ёлҗңлӢӨ.

    л ҲнҚјлҹ°мҠӨ нғҖмһ…м ҖмһҘмӢңм—җ м ҖмһҘлҗҳлҠ” нһҷ мҳҒм—ӯлҚ°мқҙн„°мқҳ лқјмқҙн”„ нғҖмһ„мқҖ к°Җл№„м§Җ м»¬л үн„°к°Җ кҙҖлҰ¬н•ңлӢӨ.

    public class example {
    public static void main(String[] args){
        String str = "Test";
        System.out.println(str);
        str = null;
        System.out.println(str);
    }
}

    strмқҖ TestлқјлҠ” нһҷмҳҒм—ӯм—җ м ҖмһҘлҗң лҚ°мқҙн„°лҘј м°ёмЎ°н•ҳлҠ” ліҖмҲҳмқёлҚ°, str=null;л¶Җл¶„мқ„ нҶөн•ҙ TestлҘј к°ҖлҰ¬нӮӨлҠ” ліҖмҲҳлҠ” лҚ”мқҙмғҒ мЎҙмһ¬н•ҳм§Җ м•ҠкІҢ лҗҳм–ҙ к°Җл№„м§Җ м»¬л үн„°к°Җ нһҷмҳҒм—ӯм—җм„ң TestлқјлҠ” лҚ°мқҙн„°лҘј м ңкұ°н•ҳкІҢ лҗңлӢӨ.

    мқҙмІҳлҹј м°ёмЎ° нғҖмһ…мқҳ нһҷ мҳҒм—ӯм—җ м ҖмһҘлҗҳлҠ” лҚ°мқҙн„°лҠ” м•„л¬ҙлҸ„ м°ёмЎ°н•ҳм§Җ м•ҠкІҢ лҗҳм—Ҳмқ„л•Ң мҶҢл©ёлҗңлӢӨ.



    нғҖмһ… ліҖнҷҳ, мәҗмҠӨнҢ… , нғҖмһ… н”„лЎңлӘЁм…ҳ

    нҠ№м • лҚ°мқҙн„° нғҖмһ…мқҳ к°’мқ„ лӢӨлҘё лҚ°мқҙн„° нғҖмһ…мқҳ к°’мқ„ ліҖнҷҳ н•ҳлҠ” кІғ

    в—ҫ нғҖмһ… н”„лЎңлӘЁм…ҳ

    мһҗмӢ мқҳ н‘ңнҳ„лІ”мң„лҘј лӘЁл‘җ нҸ¬н•Ён•ң лҚ°мқҙн„° нғҖмһ…мңјлЎң ліҖнҷҳ

    byte -> int, int-> longмҷҖ к°ҷмқҙ лҚ” нҒ° лІ”мң„лҘј к°–лҠ” лҚ°мқҙн„° нғҖмһ…мңјлЎң ліҖнҷҳмқҳ кІҪмҡ°к°Җ мҶҚн•ҳкұ°лӮҳ int -> floatмҷҖ к°ҷмқҙ к°ҷмқҖ нҒ¬кё°м—¬лҸ„ н‘ңнҳ„лІ”мң„лҘј лӘЁл‘җ нҸ¬н•Ён•  мҲҳ мһҲлӢӨл©ҙ н”„лЎңлӘЁм…ҳмқҙлӢӨ.

    float -> long мқҳ кІҪмҡ° лҚ°мқҙн„° нҒ¬кё°лЎңл§Ң ліҙл©ҙ н”„лЎңлӘЁм…ҳмқҙлқјкі  мғқк°Ғн•  мҲҳ мһҲм§Җл§Ң, longмқҖ мӢӨмҲҳлҘј м ҖмһҘн•  мҲҳ м—Ҷкё° л•Ңл¬ём—җ лҚ°мқҙн„° мҶҗмӢӨмқҙ мқјм–ҙлӮҳ н”„лЎңлӘЁм…ҳмқҙлқјкі  н•  мҲҳ м—Ҷкі  мәҗмҠӨнҢ…мқҙлқјкі  ліј мҲҳ мһҲлӢӨ.

    long l = 1234L;
float f = l;

    мң„мқҳ мҪ”л“ңмҷҖ к°ҷмқҙ н”„лЎңлӘЁм…ҳмқҳ кІҪмҡ° лҚ°мқҙн„° мҶҗмӢӨмқҙ л°ңмғқн•ҳм§Җ м•Ҡкі  мһҗлҸҷмңјлЎң нҳ•ліҖнҷҳмқ„ мӢңмјңмӨҖлӢӨ.


    в—ҫ нғҖмһ… мәҗмҠӨнҢ… : мһҗмӢ мқҳ нҸ¬нҳ„ лІ”мң„лҘј лӘЁл‘җ нҸ¬н•Ён•ҳм§Җ лӘ»н•ң лҚ°мқҙн„° нғҖмһ…мңјлЎң ліҖнҷҳ

    int->byte, long -> intмҷҖ к°ҷмқҙ нҒ° лҚ°мқҙн„°нҒ¬кё°м—җм„ң мһ‘мқҖ нҒ¬кё°мқҳ лҚ°мқҙн„° нғҖмһ…мңјлЎң ліҖнҷҳн• л•ҢлӮҳ float -> longкіј к°ҷмқҙ лҚ°мқҙн„° н‘ңнҳ„лІ”мң„к°Җ лӢ¬лқјм§ҖлҠ” кІҪмҡ°м—җ мҶҚн•ҳл©° лҚ°мқҙн„° мҶҗмӢӨмқҙ мқјм–ҙ лӮңлӢӨ.

    float f = 1.234f;
long l = f;

    мң„мқҳ мҪ”л“ңмҷҖ к°ҷмқҙ мһҗлҸҷмңјлЎң мәҗмҠӨнҢ…мқ„ н•ҳл Өкі н•ңлӢӨл©ҙ лҚ°мқҙн„° мҶҗмӢӨмқҙ л°ңмғқн•  мҲҳ мһҲкё° л•Ңл¬ём—җ м»ҙнҢҢмқјнғҖмһ„м—җ мҳӨлҘҳлҘј л°ңмғқ мӢңнӮЁлӢӨ.

    к·ёлһҳлҸ„ нҳ•ліҖнҷҳмқ„ н•ҳкі  мӢ¶лӢӨл©ҙ к°•м ңнҳ•ліҖнҷҳмқ„ мӮ¬мҡ©н•  мҲҳ мһҲм§Җл§Ң лҚ°мқҙн„° мҶҗмӢӨмқҙ л°ңмғқн•ҳкІҢ лҗңлӢӨ.

    float f = 1.234f;
long l = (long)f;
System.out.println(l); // 1



    1м°Ё, 2м°Ё л°°м—ҙ м„ м–ё

    л°°м—ҙ ліҖмҲҳ м„ м–ёкіј лҸҷмӢңм—җ мҙҲкё°нҷ”лҘј нҶөн•ҙ м„ м–ё н•ҳлҠ” л°©мӢқкіј newлҘј мқҙмҡ©н•ҳлҠ” л°©мӢқмқҙ мһҲлӢӨ.

    class ArrayExample {
	public static void main(String[] args) {
        //1м°Ёмӣҗ л°°м—ҙ
        int[] array1 = {1, 2, 3, 4, 5};
        int[] array2;
        array2 = new int[5];
        array2[0]=1;
        array2[1]=2;
        array2[2]=3;
        array2[3]=4;
        array2[4]=5;

        //2м°Ёмӣҗ л°°м—ҙ
        int[][] array3 = {{1, 2}, {3, 4}};
        int[][] array4;
        array4 = new int[2][2];
        array4[0][0] = 1;
        array4[0][1] = 2;
        array4[1][0] = 3;
        array4[1][1] = 4;

    }
}

    в—ҫ 1м°Ёмӣҗ л°°м—ҙ

    мӮ¬м§„ м¶ңмІҳ: https://www.notion.so/2-38b5d67c7f5a48238529bb8f1617ea0d


    1м°Ёмӣҗ л°°м—ҙмқҳ ліҖмҲҳ(array1,array2)лҠ” мҠӨнғқмҳҒм—ӯм—җ мЎҙмһ¬н•ҳл©° нһҷ мҳҒм—ӯмқҳ мЈјмҶҢк°’мқ„ к°–лҠ”лӢӨ.

    нһҷ мҳҒм—ӯм—җлҠ” лҚ°мқҙн„° нғҖмһ… нҒ¬кё°мҷҖ мҡ”мҶҢ к°ңмҲҳм—җ л§һкІҢ н• лӢ№лҗҳм–ҙ мӮ¬мҡ©мқҙлҗңлӢӨ.

    в—ҫ 2м°Ёмӣҗ л°°м—ҙ

    мӮ¬м§„ м¶ңмІҳ : https://www.notion.so/2-38b5d67c7f5a48238529bb8f1617ea0d


    2м°Ёмӣҗ л°°м—ҙмқҳ ліҖмҲҳ(array3,array4)лҠ” мҠӨнғқмҳҒм—ӯм—җ мЎҙмһ¬н•ҳл©° нһҷ мҳҒм—ӯмқҳ мЈјмҶҢк°’мқ„ к°–лҠ”лӢӨ.

    нһҷ мҳҒм—ӯм—җлҠ” мӢӨм ң лҚ°мқҙн„°к°’л“Өкіј 2м°Ёмӣҗл°°м—ҙмқҳ к°Ғ н–үмқҳ мӢңмһ‘мЈјмҶҢлҘј к°–лҠ” мЈјмҶҢк°’л“Өмқҙ мЎҙмһ¬н•ңлӢӨ.



    нғҖмһ… м¶”лЎ 

    Java 10 мқҙнӣ„л¶Җн„° мғқкІЁлӮң кё°лҠҘмңјлЎң м»ҙнҢҢмқјлҹ¬к°Җ к°’мқ„ ліҙкі  лҚ°мқҙн„° нғҖмһ…мқҙ л¬ҙм—Үмқём§Җ м¶”лЎ н•ҳлҠ” кІғмңјлЎң Jsмқҳ var,letмҷҖ л№„мҠ·н•ҳлӢӨ.

    в—ҫ м ңл„ӨлҰӯ

    public class example{
