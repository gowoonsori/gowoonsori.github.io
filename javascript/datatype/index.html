    мҡ”мҰҳ н•«н•ң TSк°Җ м•„лӢҢ мӣҗлһҳмқҳ кё°ліё JSлҠ” лҸҷм Ғ нғҖмһ…(лҠҗмҠЁн•ң нғҖмһ…) м–ём–ҙмқҙкё° л•Ңл¬ём—җ нғҖмһ…мқ„ лҜёлҰ¬ м„ м–ён•  н•„мҡ”к°Җ м—ҶлӢӨ.

    кё°ліё мһҗлЈҢнҳ• ( Primitive )

    в—ҫ boolean

    лӢӨлҘё м–ём–ҙл“Өкіј л§Ҳм°¬к°Җм§ҖлЎң trueмҷҖ falseк°’мқ„ к°–лҠ”лӢӨ.

    в—ҫ null

    к°’мқҙ л№„м–ҙмһҲлӢӨлҠ” кІғмқ„ н‘ңмӢңн•ҳлҠ” лҚ°мқҙн„° нҳ•мңјлЎң л№„көҗ м—°мӮ°мӢң falseлЎң м·Ёкёүн•ңлӢӨ.
    к°’мқҙ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ” лӢӨлҠ” кІғ(ліҖмҲҳк°Җ м–ҙл–Ө к°қмІҙлҸ„ к°ҖлҰ¬нӮӨкі  мһҲм§Җ м•ҠмқҢ)мқ„ мқҳлҸ„м ҒмңјлЎң к°ҖлҰ¬нӮӨкё° мң„н•ҙ мӮ¬мҡ©н•ҳкё° л•Ңл¬ём—җ typeof nullм—°мӮ°м—җ objectлҘј л°ҳнҷҳн•ңлӢӨ.

    в—ҫ undefined

    м „м—ӯ к°қмІҙмқҳ мҶҚм„ұмқ„ к°ҖлҰ¬нӮӨлҠ” к°’мңјлЎң ліҖмҲҳм—җ к°’мқ„ н• лӢ№н•ҳм§Җ м•ҠмқҖ мғҒнғңмқҙлӢӨ.

    null == undefined; //true
null === undefined; //false

    undefinedлҸ„ л№„көҗ м—°мӮ°мӢң falseлЎң м·Ёкёүн•ңлӢӨ.
    л•Ңл¬ём—җ nullкіј лҸҷл“ұ м—°мӮ°(==)мӢң trueмқҙм§Җл§Ң, м—„м—°нһҲ лӢӨлҘё нғҖмһ…мқҙкё° л•Ңл¬ём—җ nullкіј мқјм№ҳ м—°мӮ°мһҗ(===)мӢңм—җ falseк°Җ л°ҳнҷҳлҗңлӢӨ.

    в—ҫ number

    лӢӨлҘё м–ём–ҙл“ӨкіјлҠ” лӢӨлҘҙкІҢ кі м •/л¶ҖлҸҷ мҶҢмҲҳм җ л°©мӢқм—җ л”°лқј мһҗлЈҢнҳ•мқ„ лӮҳлҲ„м§Җ м•Ҡкі  нҶөмқјлҗң нҳ•нғңлЎң мӮ¬мҡ©лҗңлӢӨ.

    const bigInt = 123456789n;

    нғҖмһ…мқҳ н‘ңнҳ„ лІ”мң„лҠ” -(2^53-1) ~ 2^53-1мқҙлӢӨ.
    мқҙліҙлӢӨ лҚ” нҒ° мҲ«мһҗлҘј н‘ңнҳ„ н•ҳкі  мӢ¶лӢӨл©ҙ мҲ«мһҗ лҒқм—җ nмқ„ л¶ҷмқё BigIntнҳ•мқ„ мӮ¬мҡ©н•  мҲҳлҸ„ мһҲлӢӨ.

    Infinity

    мқҙлҰ„ к·ёлҢҖлЎң к°ҖмһҘ нҒ° к°’(вҲһ)мқ„ н‘ңнҳ„н•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳкі , лӢӨлҘё м–ём–ҙл“Өкіј лӢ¬лҰ¬ нҠ№м • мҲ«мһҗлҘј 0мңјлЎң лӮҳлҲҢ мҲҳ мһҲлҠ”лҚ° мқҙл•Ң Infinityк°’мқҙ л°ҳнҷҳлҗңлӢӨ.

    лҳҗн•ң, JsлҠ” 0мқ„ +0кіј -0 л‘җк°Җм§ҖлЎң н‘ңнҳ„н•  мҲҳ мһҲлҠ”лҚ°, лӮҳлҲ„лҠ” л¶Җнҳём—җ л”°лқј InfinityлҸ„ л¶Җнҳёк°Җ кІ°м •лҗңлӢӨ.

    NaN

    м•Ҫмһҗ к·ёлҢҖлЎң мҲ«мһҗк°Җ м•„лӢҲлқјлҠ” кІғмқ„ н‘ңнҳ„н•ҳлҠ” к°’мқҙлӢӨ.

    console.log('л¬ёмһҗм—ҙ лӮҳлҲ„кё° мҲ«мһҗ' / 2); //NaN

    в—ҫ String

    2byteмқҳ unsigned м •мҲҳк°’ л“Өмқҳ м§‘н•©мқҙлӢӨ.
    н•ңлІҲ мғқм„ұлҗҳл©ҙ immutable н•ҳлӢӨ.

    в—ҫ symbol

    мң мқјн•ҳкі  immutableн•ң мӣҗмӢң к°’мқҙкё° л•Ңл¬ём—җ, к°қмІҙмқҳ мӢқлі„мһҗ(мқјмў…мқҳ privateн•ң keyк°’)м—җ мӮ¬мҡ©лҗңлӢӨ.

    Symbol() л©”м„ңл“ңлҘј мқҙмҡ©н•ҳм—¬ кі мң н•ң мӢ¬ліјк°’мқ„ л°ҳнҷҳл°ӣмқ„ мҲҳ мһҲлӢӨ.

    Symbol('ex') === Symbol('ex'); //false
Symbol('ex') == Symbol('ex'); //false

var ex = new Symbol(); //error
Symbol(12) + 1; //error
Symbol(12) + '1'; //error
typeof Symbol() === 'symbol';

    нҠ№м§•

    • л№„м—ҙкұ°нҳ• (for…inмӮ¬мҡ© л¶Ҳк°Җ)

    • new нӮӨмӣҢл“ңлҘј м§Җмӣҗн•ҳм§Җ м•ҠлҠ”лӢӨ.

    • мҲ«мһҗлӮҳ л¬ёмһҗм—ҙлЎң нҳ•ліҖнҷҳ лҗҳм§Җ м•ҠлҠ”лӢӨ. (TypeError)

    • Object.getOwnPropertyNames() л©”м„ңл“ңм—җм„ң мӢ¬ліј мҶҚм„ұмқҖ л°ҳнҷҳлҗҳм§Җ м•ҠлҠ”лӢӨ.

      • JSON.stringify()м—җм„ңлҸ„ л¬ҙмӢңлҗңлӢӨ.

    • Object.assign() мқҖ мӢ¬ліјнҸ¬н•Ё лӘЁл“  н”„лЎңнҚјнӢ°лҘј ліөмӮ¬н•ңлӢӨ.

    • Symbol.for() мқ„ нҶөн•ҙ м „м—ӯ мӢ¬ліјмқ„ л§Ңл“Ө мҲҳ мһҲлӢӨ.


    к°қмІҙ

    let user = { name: 'hong' };
let user2 = new Object();

    мӢқлі„мһҗлҘј нҶөн•ҙ м°ёмЎ°н•  мҲҳ мһҲлҠ” Refrerence Type мқҙлӢӨ.

    м—¬лҹ¬ лҚ°мқҙн„°л“Өмқ„ лӢҙмқ„ мҲҳ мһҲлҠ” лҚ°мқҙн„°нғҖмһ…мңјлЎң мқјмў…мқҳ collection.
    keyмҷҖ valueлҘј мҢҚмңјлЎң к°–лҠ” лҚ°мқҙн„° нғҖмһ…мңјлЎң keyм—җлҠ” л¬ёмһҗм—ҙл§Ң мҳ¬ мҲҳ мһҲл”°.

    к°қмІҙ м•Ҳмқҳ лҚ°мқҙн„°лҘј ліҖнҷҳн•ҳм§Җ лӘ»н•ҳкІҢ н•ҳлҠ” мҶҚм„ұмқҙлӮҳ к°қмІҙм—җ м ‘к·јн•  мҲҳ мһҲкІҢ н•ҳлҠ” мҶҚм„ұ л‘җк°ңмқҳ мҶҚм„ұмқҙ мһҲмңјл©° л‘ҳ мӨ‘ н•ңк°ңл§Ң к°–мқ„ мҲҳ мһҲлӢӨ.

    в—ҫ лҚ°мқҙн„° мҶҚм„ұ

    к°қмІҙ лӮҙ лҚ°мқҙн„°лҘј мҲҳм •н•ҳм§Җ лӘ»н•ҳкІҢ н•ҳлҠ” мҶҚм„ұ

    • [[Writable]] : к°қмІҙм•Ҳмқҳ к°’мқҳ мҲҳм • к¶Ңн•ңмқ„ м§Җм • (trueл©ҙ мҲҳм • к°ҖлҠҘ)

    • [[Enumerable]] : м—ҙкұ°нҳ•мңјлЎң мӮ¬мҡ© к°ҖлҠҘн•ңм§Җ м§Җм • (trueл©ҙ м—ҙкұ°нҳ•мңјлЎң for…in мӮ¬мҡ©к°ҖлҠҘ)

    • [[Configurable]] : falseлқјл©ҙ Configurableкіј Enumerableмқ„ мҲҳм •мқҙ л¶Ҳк°ҖлҠҘн•ҳлӢӨ.

    в—ҫ м ‘к·јмһҗ мҶҚм„ұ

    к°қмІҙлӮҙ м ‘к·јмһҗ н•ЁмҲҳ(get/set)лҘј кё°мҲ н•ҳлҠ” мҶҚм„ұ

    • [[Get]] : к°қмІҙ лӮҙмқҳ get нӮӨмӣҢл“ңлҘј нҶөн•ҙ м •мқҳлҗң л©”м„ңл“ңлҘј нҳём¶ңн•ҳкі  к·ё л°ҳнҷҳк°’мқ„ к°Җм ёмҳЁлӢӨ.

    • [[Set]] : н• лӢ№лҗң к°’мқ„ мқёмһҗлЎң к°қмІҙ лӮҙмқҳ set нӮӨмӣҢл“ңлҘј нҶөн•ҙ м •мқҳлҗң л©”м„ңл“ңлҘј нҳём¶ң

    • [[Enumerable]] : м—ҙкұ°нҳ•мңјлЎң мӮ¬мҡ© к°ҖлҠҘн•ңм§Җ м§Җм •

    • [[Configurable]] : flaseлқјл©ҙ Configurableкіј лҚ°мқҙн„° мҶҚм„ұмқ„ мҲҳм •н•  мҲҳ м—ҶлӢӨ.


    Primitive Wrapper Object (мӣҗмӢң лһҳнҚј к°қмІҙ)

    м•һм„ң лӮҳм—ҙн•ң мӣҗмӢңк°’мқҙ л©”м„ңл“ңлӮҳ н”„лЎңнҚјнӢ°м—җ м ‘к·јн•  мҲҳ мһҲкІҢ н•ҙмЈјлҠ” нҠ№мҲҳ к°қмІҙ

    String, Number, Boolean, Symbolкіј к°ҷлӢӨ.

    let user = 'hong';
console.log(user.toUpperCase()); //HONG

    мӣҗмӢңк°’мқ„ лһҳнҚјк°қмІҙмқҳ н•ЁмҲҳлҘј мқҙмҡ©н•ҳм—¬ м ‘к·јн•ҳлҠ” мҲңк°„ лһҳнҚј к°қмІҙк°Җ мһҗлҸҷмңјлЎң л§Ңл“Өм–ҙм ё л©”м„ңл“ңлҘј мқҙмҡ©н•ҳкі  л°ҳнҷҳк°’мқ„ л°ҳнҷҳн•ң нӣ„ лһҳнҚј к°қмІҙлҠ” мһҗлҸҷ мҶҢл©ёлҗңлӢӨ.

    Array

    мҲңм„ңк°Җ мһҲлҠ” к°қмІҙлҘј н‘ңнҳ„н•ҳкі , keyлҘј л¬ёмһҗм—ҙмқҙ м•„лӢҢ м •мҲҳлҘј к°–мқ„ мҲҳ мһҲлҠ” к°қмІҙ

    let user = ['hong', 'kim'];

alert(user[1]); //kim

    лҢҖкҙ„нҳё []лЎң н‘ңнҳ„мқҙ к°ҖлҠҘн•ҳкі  keyлҘј м •мҲҳлЎң м ‘к·јмқҙ к°ҖлҠҘн•ҳлӢӨ.

    к·ё мҷё Set, Map л“ұмқҙ к°қмІҙм—җ мҶҚн•ңлӢӨ.



    нҳ•ліҖнҷҳ

    в—ҫ лӘ…мӢңм Ғ нҳ•ліҖнҷҳ

    typeof String(1); //"string"
typeof Number('1'); //"number"
typeof Boolean(0); //"boolean"

console.log(Number('12')); //12
console.log(Number('hello')); //NaN
console.log(Number(undefined)); //0
console.log(Number(NaN)); //0
console.log(Number(null)); //0

console.log(Boolean(0)); //false
console.log(Boolean(null)); //false
console.log(Boolean(undefined)); //false
console.log(Boolean(NaN)); //false

    нҳ• ліҖнҷҳмқ„ лӘ…мӢңм ҒмңјлЎң н•ҙмЈјлҠ” кІғмңјлЎң мҳҲм ңмҷҖ к°ҷмқҖ кІ°кіјк°’мқ„ к°–лҠ”лӢӨ.

    в—ҫ мһҗлҸҷ нҳ•ліҖнҷҳ

    лӘ…мӢңм ҒмңјлЎң м§ҖмӢң н•ҳм§Җ м•Ҡм•„лҸ„ лҢҖл¶Җл¶„ мһҗлҸҷнҳ•ліҖнҷҳмқ„ нҶөн•ҙ м Ғм Ҳн•ҳкІҢ ліҖнҷҳлҗңлӢӨ.

    мқҙлҘј мқҙмҡ©н•ҙ Stringмқ„ мҲ«мһҗлЎң мүҪкІҢ ліҖнҷҳ мӢңнӮ¬ мҲҳ мһҲлӢӨ.

    console.log(typeof +'11'); //number
console.log(typeof 1 * '11'); //number
console.log(typeof '11' / 1); //number

    м—°мӮ°мқ„ мҲҳн–үн•ҙлҸ„ к°’мқҙ ліҖн•ҳм§Җ м•ҠлҠ” 1мқ„ кіұн•ҳкұ°лӮҳ лӮҳлҲ„лҠ” л°©мӢқмқҙлӮҳ +м—°мӮ°мһҗлҘј л¶ҷмқҙлҠ” л°©лІ•мңјлЎң мүҪкІҢ ліҖнҷҳмқҙ к°ҖлҠҘн•ҳлӢӨ.
